# Robotické dvojča pre priemyselnú prax

Tento repozitár obsahuje zálohu programu z reálneho priemyselného robota FANUC, ktorá slúži 
na simulácie v prostredí FANUC ROBOGUIDE.

Backup z reálneho robota obsahuje zdrojové programy, registre a ďalšie dáta, ktoré je potrebné načítať do virtuálneho pracoviska v ROBOGUIDE a následne 
prispôsobiť konkrétnemu modelu bunky, nástroju a rozloženiu pracoviska.


## Cieľ :
- prácu s reálnym kódom robota mimo výroby
- vytvorenie simulácie v prostredí ROBOGUIDE
- testovanie trajektórií a logiky programu
- dolaďovanie problémových situácií bez zásahu do reálnej prevádzky

## Obsah repozitára
- programy robota vo formáte `.TP` alebo `.DF`,
- registre, pozície a konfiguračné dáta,
- pomocné súbory pre obnovenie alebo import do virtuálneho kontroléra
